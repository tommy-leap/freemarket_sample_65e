## usersテーブル
| Column               | Type    | Options     |
| -------------------- | ------- | ----------- |
| nickname             | string  | null: false |
| image                | text    |             |
| evaluation           | integer |             |
| email                | string  |             |
| encrypted_password   | string  |             |
| evaluation           | integer |             |
| reset_password_token | string  |             |
| name                 | string  |             |

### Association
- has_one :user_detail
- has_many :products
- has_many :likes
- has_many :messages
- has_many :cards
- has_many :sns_credentials
